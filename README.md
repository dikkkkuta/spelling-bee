# Spelling Bee

## Features
- Play the classic Spelling Bee game.
- Interactive gameplay with real-time feedback.
- Leaderboard to track scores.

## How to Play
1. Start the game by clicking on the 'Play' button.
2. Spell the words as they appear on the screen.
3. Try to get the highest score!
